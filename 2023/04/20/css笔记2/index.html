1.Emmet语法

快速生成html语法

  1. 标签名+Tab直接生成标签

  2. 生成多个相同标签 标签名*n div*3

  3. 父子级别关系的标签，用> , ul>li

  4. 兄弟级别的标签，用+， div+p

  5. 点+类名，生成带类名的标签；#+id名，生成带id名的标签

  6. .demo$*5,生成五个按序排列的div

div,p {
  样式声明
}
div和p都改了同一个样式

用冒号表示   :link

     a:link	没有点击过的(访问过的)链接

     a:visited	点击过的(访问过的)链接

     a:hover	鼠标经过的那个链接

     a:active	鼠标正在按下还没有弹起鼠标的那个链接
